SELL THE NEWS - MULTI-FEED MARKET DESKTOP WIDGET
================================================

PROJECT STRUCTURE:
- Models/: Data models (SellTheNewsSummary, FinancialJuiceHeadline)
- Services/: API services (SellTheNewsService, FinancialJuiceService)
- Form1.cs: Main UI (tabbed interface, 3 feeds)
- Program.cs: Entry point

FEATURES:
✓ Sell The News Feed
✓ Financial Juice Feed  
✓ Tab Navigation (3 tabs)
✓ Independent Timers (non-blocking)
✓ Error Resilience (one feed failing doesn't break others)
✓ Desktop Widget (always-on-top, draggable, compact)
✓ Dark Theme (modern UI)
✓ Full Report Display

API ENDPOINTS:
- Sell The News: https://sellthenews.org/api/wsb/latest?lang=zh
- Financial Juice: newsapi.org (test data)

WINDOW INFO:
- Size: 500x380 pixels
- Default Position: Top-right corner
- Always-on-top

REFRESH SCHEDULE:
- Sell The News: 30 seconds
- Financial Juice: 45 seconds

BUILD:
✓ .NET 10
✓ C# 14.0
✓ No external dependencies

CUSTOMIZATION:
To change Financial Juice API endpoint, edit FinancialJuiceService.cs (line 18-19)
To adjust refresh intervals, edit Form1.cs SetupTimers() method
