📊 Overview
News Widget is a lightweight Windows desktop application that displays real-time financial market analysis and news feeds. It provides traders with a minimal, always-on-top widget (500×380px) for quick market insights.
The application aggregates data from multiple sources:
•	Sell The News (STN): Complete WSB market reports with multi-section analysis
•	Financial Juice (FJ): Financial news headlines and market updates
•	Overview Dashboard: Combined summary view of all feeds
---
✨ Features
Core Functionality
•	✅ Multi-Feed Integration - Two independent data sources with separate refresh cycles
•	✅ Tabbed Navigation - Overview | Sell The News | Financial Juice tabs
•	✅ Full Report Display - Complete multi-section reports with professional formatting
•	✅ Real-Time Refresh - Automatic timers (30s STN, 45s FJ) + manual refresh button
•	✅ Error Resilience - One feed failing doesn't break the other
